# Functional Requirements

- Customers can browse Home, About, Classes/Services, Shop, Product Detail, Service Detail, Cart, Checkout, and Order Confirmation.
- Customers can register, verify email, log in, reset password, and manage profile.
- Product and service detail pages support add-to-cart.
- Cart supports quantity update and item removal.
- Checkout supports simulated PayPal and mock card paths.
- Order confirmation shows order number and payment reference.
- Contact form sends notification email.
- Admin can view dashboard summary, users, products, services, and orders.
- Admin can update order status and maintain catalog entries.
