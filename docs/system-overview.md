# Karate Skillz Dojo - System Overview

- Frontend: React + Vite + Tailwind multi-route application.
- Backend: Node.js + Express REST API (`/api/v1`).
- Auth: JWT bearer token with customer/admin role checks.
- Checkout: simulated PayPal-style and mock card payment paths.
- Email: MailEnable SMTP via Nodemailer for verification/reset/order/contact.
